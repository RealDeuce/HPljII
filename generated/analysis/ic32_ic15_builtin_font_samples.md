# IC32/IC15 Built-In Font Direct Samples

These samples consume the same mode-1 payload bytes emitted in
`ic32_ic15_builtin_glyph_payloads.json` and decode them directly into `#`/`.`
rows. They prove the extractor output is renderable font input, but they are
not a firmware cursor/baseline or self-test placement proof.
